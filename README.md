🧠 Multi-Agent AI Report Generator

This project is a Flask-based multi-agent AI system that processes user queries through multiple intelligent agents to generate structured and insightful reports.

🚀 Overview
The application uses a pipeline of three agents:
Knowledge Agent – Retrieves and summarizes relevant information
Analysis Agent – Performs deep reasoning and insights
Response Agent – Generates a clean, structured HTML report

The final output is a well-organized report that can also be downloaded.

⚙️ Features
Multi-agent architecture for better reasoning
Clean and interactive web interface using Flask
Automated report generation in HTML format
Downloadable reports for offline use
Uses OpenAI models for intelligent processing

🛠️ Tech Stack
Python
Flask
OpenAI API
HTML/CSS

▶️ How to Run
Clone the repository

Install dependencies:
pip install -r requirements.txt

Add your API key in .env:
OPENAI_API_KEY=your_api_key_here

Run the app:
python app.py

Open in browser:
http://localhost:8501

📄 Output
Executive Summary
Detailed Analysis
Recommended Next Steps

All results are displayed on the web page and can be downloaded as an HTML report.
